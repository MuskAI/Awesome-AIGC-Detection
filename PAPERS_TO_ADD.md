# Paper Backlog

This file tracks candidate papers whose metadata is not complete enough for `data/papers.json` yet.

Before moving a paper into the main data source:

- Confirm the title, year, month, area, venue, and canonical paper URL.
- Add a real `paper_url`; do not use placeholders such as `https://arxiv.org/abs/`.
- Add `arxiv_id` only when the paper has a valid arXiv identifier.
- Add `code_url` or `project_url` only when the target URL is confirmed.
- Run `python3 scripts/validate_data.py` and `python3 scripts/generate_readme.py`.
